# QA_CHECK

Проверки для текущей GitHub-сборки.

## Сделано

- Монограмма ХВ вынесена в SVG-пути, а не набрана шрифтом.
- Нижняя линия у монограммы не используется.
- У SVG viewBox есть запас по низу, поэтому финальная ХВ не должна обрезаться.
- Декоративная полоска находится внутри зеленого hero-блока.
- Иконка колец нарисована как line icon с камнями и пересечением, не как два пустых круга.
- Иконки программы стоят справа от текста.
- Цвет имени гостя совпадает с цветом слова «Дорогие».
- Случайные сердечки и звездочки не добавлены.
- FAQ не содержит вопрос про подтверждение участия.
- В программе не использованы нежелательные формулировки про финал программы и бронь зала.
- Поиск по файлам не нашел букву е.

## Что проверить глазами после публикации

- Шапка на телефоне: ХВ слева, дата по центру, бургер справа.
- Первый зеленый экран: полоска находится между шапкой и карточкой.
- Финальный экран: большая ХВ не обрезана снизу.
- Ссылки `?guest=...` корректно подставляют имя гостя.
- Для `child_demo` анкета скрыта.
- Для `banquet_demo` показана только банкетная часть программы.
