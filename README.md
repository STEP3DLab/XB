# Свадебное приглашение ХВ

Сайт опубликован через GitHub Pages: https://step3dlab.github.io/XB/

## Персональные ссылки гостей

Ссылки формируются через параметр `guest`:

```text
https://step3dlab.github.io/XB/?guest=momanddad
```

Список гостей и ID хранится в Google Sheets в листе `Гости`.

## Анкета по напиткам

Форма на сайте готова. Для автоматической записи ответов в Google Sheets нужно развернуть Apps Script:

1. Открыть таблицу `Приглашение на свадьбу`.
2. Расширения → Apps Script.
3. Вставить код из `apps-script.js`.
4. Развернуть → Новое развертывание → Веб-приложение.
5. Execute as: Me. Who has access: Anyone.
6. Скопировать Web App URL и вставить его в `index.html` в константу `API_URL`.

До подключения `API_URL` форма сохраняет ответ локально в браузере гостя и показывает техническое сообщение.
