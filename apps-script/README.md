# Apps Script для свадебного сайта

## Подключение

1. Откройте созданную Google-таблицу.
2. Выберите **Расширения → Apps Script**.
3. Удалите стандартный код и вставьте содержимое файла `Code.gs`.
4. Сохраните проект.
5. Один раз запустите функцию `setupWeddingWorkbook` и подтвердите разрешения.
6. Выберите **Развернуть → Новое развертывание → Веб-приложение**.
7. Параметры:
   - выполнять от имени: **я**;
   - доступ: **все, у кого есть ссылка**.
8. Скопируйте URL веб-приложения.
9. В репозитории откройте `config.js` и вставьте URL:

```js
window.XB_APPS_SCRIPT_URL = 'ВАШ_URL';
```

После этого сайт будет:

- получать обращение и сценарий из листа `Гости`;
- показывать нужные блоки;
- записывать подтверждения на лист `Ответы`.

## Добавление нового гостя

На листе `Гости` добавьте строку:

- `id` — латиницей без пробелов, например `ivan-maria`;
- `Обращение` — `Дорогие`, `Дорогая` или `Дорогой`;
- `Имя` — текст, который увидит гость;
- `Сценарий` — один из вариантов листа `Сценарии`;
- в override-колонках обычно оставляйте `По сценарию`;
- `Активно` — `Да`.

Ссылка появится в колонке `Ссылка`. При необходимости выберите меню **Свадебный сайт → Обновить ссылки гостей**.
